# Scooter Plus - Landing Page

Landing page untuk Scooter Plus Garage, servis Vespa matic premium di Jakarta.

## Tech Stack

- **React 18** - UI Library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Redux Toolkit** - State management
- **React Router** - Routing
- **Framer Motion** - Animations
- **Lucide React** - Icons
