## Tips Download:
- Gunakan format PNG dengan background transparan
- Ukuran minimal 400x400px untuk kualitas yang baik
- Pastikan logo terpusat dan jelas
- Component akan otomatis membuat bentuk lingkaran

## Cara Download:
1. Buka Google Images
2. Cari "Nama Brand logo PNG transparent"
3. Pilih gambar dengan resolusi tinggi dan background transparan
4. Download dan simpan dengan nama yang sesuai di folder ini
5. Setelah semua logo di-download, update file `Home.jsx` untuk menggunakan import lokal:
   ```javascript
   import motulLogo from '../assets/partners/motul.png';
   import ravenolLogo from '../assets/partners/ravenol.png';
   // dst...
   ```

## Catatan:
Saat ini kode menggunakan URL logo dari CDN sebagai placeholder. Setelah download logo lokal, ganti dengan import statement di `Home.jsx`.
